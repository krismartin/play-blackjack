# Play Blackjack

A classic game, made to learn Elm.

## Getting Started

1. [Install Elm](https://guide.elm-lang.org/install/elm.html)
2. Install `elm-live`: `npm install -g elm-live`
3. Run `bin/start` to start a dev server
